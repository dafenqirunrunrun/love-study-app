# 考研伴侣 (Love & Study) - 运行指南

## 🚀 运行步骤

### 1. 安装依赖
```bash
npm install
```

### 2. 启动开发服务器
```bash
npm run dev
```

### 3. 打开浏览器
访问 http://localhost:3000 即可看到应用

## 📁 项目结构
```
love-study-app/
├── index.html          # HTML 入口文件
├── package.json        # 项目配置和依赖
├── vite.config.js      # Vite 构建配置
├── tailwind.config.js  # Tailwind CSS 配置
├── postcss.config.js   # PostCSS 配置
└── src/
    ├── main.js         # Vue 应用入口
    ├── App.vue         # 主应用组件
    └── style.css       # 全局样式
```

## 🎯 功能特色
- ✅ 考研倒计时（2026-12-26）
- ✅ 每日暖心鼓励语
- ✅ 番茄钟专注计时器（25分钟）
- ✅ 任务管理系统
- ✅ 恋爱积分奖励系统
- ✅ 心愿单兑换功能
- ✅ LocalStorage 数据持久化

## 🛠️ 技术栈
- Vue 3 (Composition API)
- Tailwind CSS
- Vite
- 纯前端，无后端

## 📱 响应式设计
应用完美适配桌面和移动设备！
