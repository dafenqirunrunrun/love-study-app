# 学习伴侣 (Study Companion) - 全能学习助手

一个功能丰富的全能学习助手，帮助你规划学习、保持专注、养成习惯、达成目标！

## 🚀 运行步骤

### 1. 安装依赖
```bash
npm install
```

### 2. 启动开发服务器
```bash
npm run dev
```

### 3. 打开浏览器
访问 http://localhost:3000 即可看到应用

## 🎯 功能特色

### 📊 首页与统计
- ✅ **通用倒计时** - 自定义目标名称和日期（考试、面试、答辩等）
- ✅ **每日暖心鼓励语** - 积极向上的学习动力
- ✅ **学习数据统计仪表盘** - 可视化学习进度
- ✅ **日历视图** - 查看每日完成情况

### 📝 任务管理
- ✅ **任务管理系统** - 支持分类（英语、政治、数学、专业课等）
- ✅ **优先级设置** - 高/中/低优先级
- ✅ **智能筛选** - 按状态、分类、日期筛选
- ✅ **批量操作** - 批量标记完成/删除
- ✅ **任务备注** - 记录详细学习内容

### ⏱️ 番茄专注
- ✅ **番茄钟专注计时器** - 可自定义时长（1-60分钟）
- ✅ **短休息/长休息** - 科学的时间管理
- ✅ **专注历史记录** - 追踪每日专注数据
- ✅ **白噪音环境音** - 多种背景音效可选
- ✅ **音效提示** - 专注完成提醒

### 📅 习惯打卡
- ✅ **习惯打卡系统** - 每日习惯追踪
- ✅ **连续打卡统计** - 记录最长连续天数
- ✅ **打卡历史** - 查看每日完成情况

### 💎 积分中心
- ✅ **个人积分系统** - 完成任务、打卡、专注获得积分
- ✅ - 新手→ **等级体系**学者→学霸→大咖→传奇
- ✅ **虚拟宠物** - 学习伴侣小助手
- ✅ **20种可兑换奖励** - 从冰淇淋到环球旅行

### 🏆 成就系统
- ✅ **50个成就徽章** - 涵盖学习各方面
- ✅ **任务成就** - 完成1/10/50/100/200/500/1000个任务
- ✅ **专注成就** - 10分钟~500小时专注时长
- ✅ **打卡成就** - 3天~365天连续打卡
- ✅ **积分成就** - 100~10000积分
- ✅ **心得成就** - 撰写学习心得
- ✅ **里程碑成就** - 累计学习7/30/100/365天
- ✅ **特殊成就** - 早起鸟儿、夜猫子等

### 📚 学习计划
- ✅ **AI学习计划生成器** - 智能规划学习路径
- ✅ **目标设置** - 自定义考试/目标日期
- ✅ **科目选择** - 英语、政治、数学、专业课等
- ✅ **强弱项分析** - AI智能分配时间
- ✅ **学习偏好设置** - 时段偏好、休息间隔
- ✅ **每周模板生成** - 智能生成学习计划

### 💭 学习心得
- ✅ **每日心得记录** - 记录学习感悟
- ✅ **心情追踪** - 记录每日心情
- ✅ **历史回顾** - 查看过往心得

### ⚙️ 设置与数据
- ✅ **通用倒计时设置** - 自定义目标名称和日期
- ✅ **深色/浅色模式** - 日间夜间护眼
- ✅ **番茄钟设置** - 自定义专注时长和休息时间
- ✅ **通知设置** - 任务、专注、成就提醒
- ✅ **数据导出/导入** - 数据备份与恢复
- ✅ **初始化功能** - 重置学习数据
- ✅ **LocalStorage持久化** - 数据自动保存

## 🎨 设计特点

### 视觉设计
- **玻璃拟态（Glassmorphism）风格** - 时尚现代的视觉效果
- **动态渐变背景** - 丰富色彩层次
- **粒子动画效果** - 生动的页面元素
- **流畅的过渡动画** - 平滑的用户体验
- **日间/夜间模式** - 完美适配各种环境

### UI组件
- **骨架屏加载** - 优雅的加载状态
- **卡片式布局** - 清晰的信息展示
- **响应式设计** - 完美适配桌面和移动端
- **触摸优化** - 移动端友好交互

## 🛠️ 技术栈

- **Vue 3** (Composition API)
- **Vue Router 4** (SPA路由)
- **Tailwind CSS** (原子化CSS)
- **Vite** (快速构建工具)
- **TypeScript** (类型安全)
- **LocalStorage** (数据持久化)
- **纯前端** - 无需后端服务
